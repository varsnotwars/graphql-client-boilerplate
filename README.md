### Features:

- Signup
- Login
- Logout (from single client or all clients)
- User confirmation
- Forgot password
- Sessions with cookies
- Authentication middleware
- Email sending (user confirmation, password resets)

### Uses:

- react hooks
- apollo boost
- reactstrap
